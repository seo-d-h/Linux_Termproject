### 3) S/W 개발환경
|**구분**|**항목**|**적용내역**|
|:---:|:---:|:---:|
|OS|Linux|시스템 환경|
|개발도구|vi editor|코드 편집기|
|컴파일러|GNU(gcc)|C코드 컴파일|
|개발언어|C|하드웨어 제어|

### 4) H/W 개발환경
|**구분**|**항목**|**적용내역**|
|:---:|:---:|:---:|
|디바이스|RaspberryPi 4B|OS를 구동하기 위한 머신|
|통신|Bluetooth|스마트폰과 보드간의 블루투스 통신|

### 5) 장비(기자재/재료) 활용
|**품명**|**작품에서의 주요기능**|
|:---:|:---:|
|Cds cell|외부조도 입력|
|SG-90(Servo  Motor)|전등제어|
|HC-06(Bluetooth)|스마트폰과 블루투스 통신|
|LCD디스플레이(I2C)|조명모드와 상태를 LCD를 통해 알려준다|
